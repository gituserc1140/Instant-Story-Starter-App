# Instant Story Starter

A Streamlit micro-app that generates a single atmospheric opening line for a story using the Groq API.

## Features
- Generate a cinematic, intriguing opening line with one click.
- Optional genre and tone selection (Mystery, Sci-Fi, Fantasy, Romance, Surreal, Dramatic).
- Cooldown mechanism to prevent spam.
- Clean, minimal, and fast UI.

## Setup
1. Install required packages:
   ```bash
   pip install -r requirements.txt
   ```
2. Replace `your_groq_api_key_here` in `app.py` with your actual Groq API key.
3. Run the app:
   ```bash
   streamlit run app.py
   ```

## Usage
- Select a genre and tone (optional).
- Click the "Generate Story Starter" button.
- Wait for the atmospheric opening line to appear.

Enjoy sparking your creativity!
